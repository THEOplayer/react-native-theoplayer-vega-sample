## CLI Commands

| Command                                                | Purpose                                  |
|--------------------------------------------------------|------------------------------------------|
| `kepler device simulator [start \| stop \| status]`    | Start, stop or request simulator status. |
| `kepler device shell -d Simulator`                     | Open simulator shell                     |
| `kepler device list` or `vda devices -l`               | List available devices                   |
| `kepler build -b Debug`                                | Build package in debug mode              |
| `vlcm list` (in shell)                                 | List available packages                  |
| `kepler device launch-app`                             | Launch app                               |
| `kepler device terminate-app`                          | Terminate app                            |
| `kepler device uninstall-app`                          | Uninstall app                            |
| `vda shell cat /etc/os-release \| grep VERSION_NUMBER` | Get device version                       |
